    Project-1

    Blink-LED💡

    Components used in this project:

    1. Arduino RoboBlack/UNO
    2. BreadBoard
    3. Jumper wires
    4. LED
    5. 220ohm resistor

    NOTE: what you need to know that every electronic device is made up of software and hardware. Connecting the components may seem easy, but the software behind it helps you understand how the computer controls the electronic components.

    1st step: before connecting you need to know that there are two types of connection.

    • Active-HIGH
    • Active-LOW

    Arduino Sketch (a code for programming the Arduino board in C++ language).

    
		// Blink built-in LED

void setup(){
  pinMode(LED_BUILTIN, OUTPUT);   // Configure built-in LED pin as output
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);  // Turn LED on
  delay(1000);                      // Wait 1 second (1000 ms)
  digitalWrite(LED_BUILTIN, LOW);   // Turn LED off
  delay(1000);                      // Wait 1 second
}
